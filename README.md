# E-Commerce-Sales-KPI-Dashboard
This project presents an interactive E-Commerce KPI Dashboard designed to monitor key business performance metrics and help data-driven decision-making. The dashboard provides a comprehensive view of sales performance, customer behavior, and product insights by leveraging Excel Pivot Tables, Slicers, and dynamic GETPIVOTDATA formulas.
